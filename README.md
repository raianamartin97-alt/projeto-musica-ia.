## 📝 Descrição do Projeto

Este projeto explora a união entre criatividade humana e Inteligência Artificial na composição musical.

Utilizando **Gemini Gems** para construção de prompts e geração criativa de letras, e o **Suno AI** para produção musical, foi desenvolvido um EP experimental com 5 faixas inéditas.

O objetivo foi demonstrar como ferramentas de IA generativa podem ser aplicadas no processo criativo musical, desde a concepção da letra até a produção final.

---

## 🛠️ Tecnologias Utilizadas

- **Gemini Gems** → criação e refinamento das letras  
- **Suno AI** → geração musical e produção de áudio  
- **GitHub** → documentação e versionamento  
- **Prompt Engineering** → modelagem criativa  

---

## 🚀 Objetivos do Projeto

- Explorar IA aplicada à música
- Desenvolver habilidades em prompt engineering
- Experimentar geração criativa com modelos generativos
- Documentar um fluxo completo de produção assistida por IA

## 📄 Processo de Criação

### 1. Planejamento
Definição do conceito musical e da proposta experimental.

### 2. Engenharia de Prompt
Criação de instruções específicas no Gemini Gems para orientar a geração das letras.

### 3. Geração Criativa
Produção das letras utilizando IA generativa.

### 4. Produção Musical
Conversão das letras em músicas completas usando Suno AI.

### 5. Organização e Documentação
Estruturação do projeto no GitHub.
